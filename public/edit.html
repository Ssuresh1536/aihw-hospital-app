API Description

This application communicates with a RESTful API hosted on Vercel and backed by MongoDB Atlas. It provides the following endpoints:

  • GET /api/pharmacies - Retrieve all pharmacy records
  • POST /api/pharmacies - Add a new pharmacy record
  • PATCH /api/pharmacies/[name] - Update selected fields (e.g., Mon-Fri, Sat, Sun-Ph)
  • DELETE /api/pharmacies/[name] - Delete a pharmacy

All requests return JSON responses and interact with a live database. Admin access is required for modifying or deleting data.
