API Description

This application communicates with a RESTful API hosted on Vercel and backed by MongoDB Atlas. It provides the following endpoints:

  • GET /api/hospitals – Retrieve all hospital records
  • POST /api/hospitals – Add a new hospital record
  • PATCH /api/hospitals/[code] – Update selected fields (resus, emergency, urgency)
  • DELETE /api/hospitals/[code] – Delete a hospital by code

All requests return JSON responses and interact with a live database. Admin access is required for modifying or deleting data.

Legal Disclaimer

This application uses publicly available hospital data from the Australian Institute of Health and Welfare (AIHW).

Data accuracy is based on the source API at the time of retrieval. The app is for demonstration or internal use and is not an official product.

No medical decisions should be made based on the displayed data. For official health information, consult WA Health or AIHW directly.
